== Installation ==

1. Upload the entire `hnp-empty` folder to the `/wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin by navigating to the #Plugins# screen in your WordPress admin area (#Plugins > Installed Plugins#).

After activation, you will see a new **HNP Empty Plugin** menu item in your WordPress admin screen.
